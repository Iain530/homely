About Homely

Autocomplete

Autocomplete results are provided using Google autocomplete.

Weather

Weather information provided using the MetaWeather api and localised to you using your devices location (if you allow it).

Icons

Icons were acquired from multiple sources:

  • GitHub for official Github logos.
  • MetaWeather for weather icons.
  • Flaticon (Author: Google) under CC 3.0 license for other icons.
  • If bookmarks are enabled in settings then icons are provided using besticon. (This is not used for top sites)

Quotes

Quotes are provided by They Said So
